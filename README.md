# Number Guessing Game 🎮

A simple, interactive command-line game where the player tries to guess a randomly generated number between 1 and 100.

## 📝 Description
This project is a classic logic game designed to practice fundamental programming concepts like loops, conditionals, and random number generation. The program picks a number, and the player receives feedback on whether their guess is "Too High," "Too Low," or "Correct!"

## 🚀 Features
*   Random number generation between 1 and 100.
*   Interactive user input via terminal/command prompt.
*   Real-time feedback on guesses.
*   Score tracking (number of attempts).

## 📖 How to Play
1.  The computer will pick a secret number between 1 and 100.
2.  Type your guess into the terminal.
3.  The program will tell you if your guess is too high or too low.
4.  Keep guessing until you find the secret number!
